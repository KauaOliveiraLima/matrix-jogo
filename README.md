# 🟢 Matrix — O Jogo de Texto

Um jogo de aventura em texto inspirado no filme *Matrix*, desenvolvido em Python.

## 📖 Sobre

Você é recrutado por Morpheus e precisa tomar decisões que vão definir o seu destino dentro da Matrix. O jogo conta com duas missões, sistema de força e saúde, e uma narrativa fiel ao universo do filme.

## 🎮 Como jogar

```bash
python matrix.py
```

Não é necessário instalar nenhuma biblioteca externa — o jogo usa apenas módulos da biblioteca padrão do Python.

## ⚙️ Requisitos

- Python 3.6 ou superior

## 🗂️ Estrutura do código

| Função               | Descrição                              |
|----------------------|----------------------------------------|
| `apresentacao()`     | Coleta o nome do jogador               |
| `conversa_morpheus()`| Primeira escolha: mundo real ou não    |
| `escolha_pilula()`   | Pílula azul ou vermelha                |
| `fuga()`             | Sistema de combate com força e saúde   |
| `descoberta_escolhido()` | Morpheus revela o destino do jogador |
| `visita_oraculo()`   | Segunda missão com o Oráculo           |

## 🚀 Funcionalidades

- Efeito de digitação cinematográfico
- Sistema de força e saúde com barra visual
- Múltiplos finais possíveis
- Código organizado em funções
